1. You are given a number `n`, representing size of array `a`.
2. You are given `n` numbers representing elements of array `a`.
3. You are required to print the next greater element on the right for all elements of array.

"Next greater element on the right" of an element x is defined as the first element on the right of x having value greater than x.
Note, if an element does not have any element on its right side greater than it, consider -1 as its "next greater element on right".

For example:

For the array [2 5 9 3 1 12 6 8 7]
Next greater for 2 is 5.
Next greater for 5 is 9.
Next greater for 9 is 12.
Next greater for 3 is 12.
Next greater for 1 is 12.
Next greater for 12 is -1.
Next greater for 6 is 8.
Next greater for 8 is -1.
Next greater for 7 is -1.
