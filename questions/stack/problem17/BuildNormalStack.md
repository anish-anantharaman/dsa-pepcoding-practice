1. You are required to create a Custom stack, which should mimic the behaviour of java.util.Stack class and implement 
LIFO semantic.
2. Here is the list of functions supposed to implement.
   <br/>
    a) push → should accept new data if there is space available in the underlying array or print "Stack overflow" 
      otherwise.
   <br/>
    b) pop → should remove and return last data if available or print "Stack underflow" otherwise.
   <br/>
    c) top → should return the last data if available or print "Stack underflow" otherwise and return -1.
   <br/>
    d) size → should return the number of elements available in the stack.
   <br/>
    e) display → should print the elements of stack in LIFO manner (space-separated) ending with a line break.
   <br/>

```
Note: Data will be ony Integers.
```

Sample input:

```
5
push 10
display
push 20
display
pop
```
