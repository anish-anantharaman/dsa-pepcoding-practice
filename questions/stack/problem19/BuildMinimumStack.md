1. As data members you have two stacks available in the class - one for data values and another one for minimum values.  
2. Here is the list of functions that you are supposed to complete:
   a) push → Should accept new data in LIFO manner.
   b) pop → Should remove and return data in LIFO manner. If not available, print "Stack underflow" and return -1.
   c) top → Should return data in LIFO manner. If not available, print "Stack underflow" and return -1.
   d) size → Should return the number of elements available in the stack.
   e) min → should return the smallest element available in the stack. If not available, print 
"Stack underflow" and return -1.
