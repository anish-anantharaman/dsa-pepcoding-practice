1. As data members you have one stack and a min element available in the class. This problem is similar to Build 
Minimum Stack in the previous problem, except that there will only be one stack and a min variable.
2. Here are the list of functions you are supposed to complete:
    a) push → should accept new data in LIFO manner.
    b) pop → should remove and return data in LIFO manner. If not available, print "Stack underflow" and return -1.
    c) top → should return data in LIFO manner. If not available, print "Stack underflow" and return -1.
    d) size → should return the number of elements available in the stack.
    e) min → should return the smallest element available in the stack. If not available, print "Stack underflow" 
      and return -1.
